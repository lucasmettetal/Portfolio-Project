Security

Production-grade security

Security was a core concern throughout — not an afterthought.

Password hashing

bcrypt with cost factor 12 — 4096 rounds, unique salt per user

JWT Authentication

Stateless tokens signed with a 32+ character secret, 7-day expiry

Role-based access

USER and ADMIN roles enforced via middleware on every protected route

Stripe webhook

Signature verified with constructEvent() — rejects any forged request

Rate limiting

10 attempts / 15 min on auth routes · 5 messages / hour on contact

Stock integrity

Atomic Prisma transaction + optimistic lock prevents overselling

HTTP headers

helmet — X-Frame-Options, HSTS, X-Content-Type-Options and more

File uploads

MIME type filter + extension allowlist via Multer
